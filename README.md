# Personalised Learning Companion

## Overview
Personalised Learning Companion is an adaptive learning platform designed to help students choose subjects, track learning progress, and explore career guidance.

## Features

• Subject Enrollment System  
• Career Guidance System  
• Assessment Quiz  
• Skill Level Detection  
• Learning Roadmaps  
• Career Videos  
• Dashboard Progress Tracking  
• Help Desk Support

## Technologies Used

Frontend:
- React.js
- TypeScript
- Tailwind CSS

Backend:
- Supabase

Tools:
- GitHub
- VS Code

1. Authentication Module
